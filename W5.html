Capstone Project Week 5


Team work

We have one more week to complete our prototype, and we are very close to finishing it. This week, we focused on some minor parts and merged our works. Stel completed the machine running feature. Josh worked on the Input System of both the keyboard and XBox controller. Dorian kept focus on Audio System. Ethan improved the art assets, and I polished the localization system and worked on the UI.


My work

This week, I worked on improving the localization system. I made a system that even if the content of a game expands to a big one, the developers could easily add more content to the script in both languages (English and French).



During the meeting with the mentors, I received some suggestions which I thought were very practical. Inspired by their suggestions, I stored the content in external files and created a ReadFile function to read the content in the txt file and added it to the dictionary. This allows the programmers can get the content by calling the GetString function with the key, and the content editors can edit and expand in the txt file.



I also added a new feature to the system which allows it to read the variables stored outside of the functions.

In addition, I improved our UI design. I used some materials from the website Kenney to change the appearance of our menu.
