Capstone Project Week 6


Team work

After completing the prototype, we started refining our game. My teammate Ethan wanted to do more programming, so we agreed with me to help build art assets and design 6 machines for our game. I see this as a good opportunity to practice my people skills, as building art assets in a continuous style requires a lot of communication. I hope I can hone my skills in the process.



At the same time, I have a task to build the lobby scene where players can join the game. I had to figure out a way to add players to the game, and this week I needed to work with my teammate Joshua as he was working on the input system.

My work

I created a scene and some hints so the player knows how to join the game.



I have an array of game objects and a public static integer to keep track of how many players will join the game.



This integer will be read in the next scene (i.e. the game scene), so it knows how many players will be set active.


Regarding the game art assets, I've done a cutout and I'll be discussing styles with Ethan. Once I have some conclusions, I start creating art assets.
